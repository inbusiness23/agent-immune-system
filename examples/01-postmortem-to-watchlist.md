# Example: Route A Postmortem To Watchlist

User complaint:

```text
The agent forgot to verify browser proof after a UI change.
```

Finding:

One missed verification is useful evidence, but not enough to create a permanent global rule.

Recommendation:

```text
Route to watchlist for the next 3 frontend review cycles.
```

Why:

The issue is real but may be context-specific. Temporary inflammation is enough unless it recurs.
