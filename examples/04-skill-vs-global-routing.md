# Example: Route To Skill Instead Of Global Instructions

Finding:

A browser proof workflow failed because a mobile artifact was missing.

Bad promotion:

```text
Add a global rule requiring mobile proof for every visual artifact.
```

Better route:

```text
Keep the rule inside the browser-proof or visual-review skill.
```

Why:

The lesson is useful, but narrow. Global memory would increase token cost and over-trigger outside visual review work.
