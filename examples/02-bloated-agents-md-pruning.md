# Example: Prune A Bloated AGENTS.md

User complaint:

```text
My AGENTS.md is huge and agents keep getting stuck asking permission.
```

Finding:

Several rules were created from narrow incidents and now suppress safe investigation.

Recommendation:

```text
Demote three incident-specific rules into dormant memory and replace them with one proportionality principle.
```

Outcome:

The active instruction surface shrinks while the historical evidence remains available if the pattern recurs.
